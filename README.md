# C++ Data Structures

This repository contains practical implementations of fundamental Data Structures concepts using C++ and Object-Oriented Programming (OOP).

## Topics Covered

* Arrays
* Matrix Operations
* Singly Linked List
* Recursive Linked List
* Doubly Linked List
* Stack using Array
* Stack using Linked List
* Queue using Array
* Queue using Linked List
* Circular Queue
* Binary Search Tree

## Features

* Object-Oriented Programming (OOP)
* Menu-driven console applications
* Searching and sorting operations
* Dynamic memory allocation
* Recursive implementations

## Technologies

* C++
* OOP Concepts

## Purpose

This repository was created for educational and training purposes while learning and teaching Data Structures concepts.
